# Script loader dofus retro

- Atualização 19/10/22

Como sabem, o objetivo da publicação do loader no github era impedir o comércio ilegal desta ferramenta que foi construída por terceiros.

Nesta atualização houveram modificações que inutilizaram o Core do Loader que estava sendo utilizado indevidamente até o momento.

Assim como fiz para consertar a função do Join na Att passada, seria possível reescrever as linhas de código para reativação dos scripts no Loader, porém, percebo que esta seria uma grande oportunidade de parar de uma vez por todas(ou pelo menos momentaneamente) com o uso desta ferramenta.

Lembrando que caso o Loader modificado volte a ser comercializado eu muito provavelmente voltarei a disponibiliza-lo de maneira gratuita, então peço que não olhem para essa pausa nas atualizações como uma oportunidade de negócios. 
FAQ: 
1:Você vai passar a vender o Loader agora?
Não. No máximo poderia ensinar a modificá-lo você mesmo, porém, será necessário conhecimentos avançados de lógica de programação e um incentivo. 
2:Qual o motivo de parar as atualizações?
Como mencionado anteriormente, esta parece ser uma boa oportunidade de encerrar o uso desta ferramenta.
Se não existe venda de terceiros, não existe motivo para atualiza-lo visto que eu não pretendo praticar venda de código alheio.
(Como se sentiriam se tivessem criado algo difícil de fazer e um de seus clientes começasse a vender pela metade do preço?).
